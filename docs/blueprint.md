# **App Name**: AutoHaus Pro

## Core Features:

- Service Listing: عرض الخدمات: الميكانيكا، إصلاح المحركات، إصلاح علبة التروس، الإصلاحات الكهربائية، برمجة الكمبيوتر، أعمال الهيكل، الطلاء، والصيانة.
- AI-Powered Visuals: إنشاء صور فريدة من نوعها تم إنشاؤها بواسطة الذكاء الاصطناعي لكل قسم خدمة لتحسين الموقع بشكل مرئي. ستستخدم كل أداة ذكاء اصطناعي المنطق عند إنشاء صور مرئية مناسبة.
- Body Work Slider: تنفيذ شريط تمرير في قسم أعمال الهيكل والطلاء لعرض صور ما قبل وبعد.
- Brand Showcase: عرض شعارات علامات السيارات الأوروبية على الصفحة الرئيسية.
- Service Details: أوصاف تفصيلية للخدمات لكل قسم من أقسام الموقع.
- Floating Contact Buttons: تنفيذ أزرار WhatsApp ومكالمات عائمة، بما في ذلك نص بجوار كل منها.

## Style Guidelines:

- اللون الأساسي: أزرق داكن (#1E3A8A) لنقل الثقة والخبرة في مجال العناية بالسيارات الأوروبية.
- لون الخلفية: رمادي فاتح (#F3F4F6) لخلفية نظيفة واحترافية.
- لون التمييز: أصفر فاتح (#FACC15) لتمييز الأزرار المهمة ودعوات الإجراء.
- خط النص الأساسي والعناوين: 'Inter'، وهو خط sans-serif ذو شكل قوطي.
- استخدام أيقونات حديثة وبسيطة يسهل التعرف عليها لتمثيل كل نوع خدمة.
- ضمان تصميم متجاوب يتكيف بسلاسة مع أجهزة سطح المكتب والأجهزة المحمولة.
- دمج انتقالات سلسة ورسوم متحركة دقيقة عند تفاعلات المستخدم.
