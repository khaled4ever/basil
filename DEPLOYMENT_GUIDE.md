# دليل نشر موقع صيانة السيارات الأوروبية

هذا الموقع مبني باستخدام إطار عمل **Next.js** ويستخدم تقنيات **Genkit** للذكاء الاصطناعي.

## خيارات النشر (Deployment Options)

### 1. الخيار الموصى به والأسهل: Firebase App Hosting
بما أنك تستخدم أدوات Firebase، فإن هذا الخيار هو الأنسب لتطبيقات Next.js الحديثة:

**الخطوات:**
1. ارفع الكود الخاص بك إلى مستودع (Repository) على **GitHub**.
2. اذهب إلى [Firebase Console](https://console.firebase.google.com/).
3. اختر مشروعك، ثم من القائمة الجانبية اختر **App Hosting**.
4. اضغط على **Get started** واربط حسابك بـ GitHub.
5. اختر المستودع والفرع (Branch) الذي تود نشره.
6. **ملاحظة هامة جداً**: لكي يعمل الذكاء الاصطناعي في الموقع المنشور:
   - بعد انتهاء النشر الأول، اذهب إلى لوحة تحكم App Hosting.
   - اذهب إلى **Settings** ثم **Environment variables**.
   - أضف متغيراً جديداً باسم `GOOGLE_GENAI_API_KEY` وضع فيه مفتاح الـ API الخاص بك.

### 2. النشر على Vercel
- اربط حساب GitHub بـ [Vercel](https://vercel.com/).
- سيقوم Vercel بالتعرف على مشروع Next.js ونشره تلقائياً.
- لا تنسَ إضافة `GOOGLE_GENAI_API_KEY` في إعدادات "Environment Variables" في Vercel.

### 3. النشر على استضافة Namecheap (تحتاج VPS)
**ملاحظة هامة:** الاستضافة المشتركة (Shared Hosting) العادية لا تدعم هذا المشروع. يجب أن يكون لديك **VPS** بنظام Linux.

**الخطوات على خادم VPS:**
1. قم بتثبيت Node.js (إصدار 18 أو أحدث).
2. ارفع ملفات المشروع إلى الخادم.
3. نفذ الأوامر التالية:
   ```bash
   npm install
   npm run build
   npm start
   ```
4. استخدم `pm2` لضمان بقاء الموقع يعمل في الخلفية.

## ملاحظة حول الصور والذكاء الاصطناعي
تأكد دائماً من إضافة مفتاح API لكي تعمل الشروحات والصور التلقائية في النسخة المباشرة للموقع.
